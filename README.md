# Movie Tracker — CC2

## Lancement

Deux terminaux nécessaires :

**Terminal 1 — API :**

```bash
pnpm install
pnpm api
```

**Terminal 2 — Application :**

```bash
pnpm dev
```

Ouvrir http://localhost:5173
